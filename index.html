🐙 GitHub Pages Setup Guide

Deploy Home Grown to GitHub Pages

GitHub Pages is the perfect free hosting solution for this site. Your website will be live at https://yourusername.github.io/home-grown — for free, forever.

1Create a GitHub Account

Go to github.com and sign up for a free account if you don't have one.

2Create a New Repository

  1. Click the green New button on your GitHub dashboard
  2. Name it home-grown
  3. Set it to Public
  4. Click Create repository

3Upload Your Website File

  1. In your new repo, click uploading an existing file
  2. Drag and drop index.html (the file you downloaded)
  3. Make sure it is named exactly index.html
  4. Click Commit changes

4Enable GitHub Pages

  1. Go to your repo → Settings tab
  2. Scroll to Pages in the left sidebar
  3. Under Source, select Deploy from a branch
  4. Choose main branch, / (root) folder
  5. Click Save

5Your Site is Live! 🎉

After ~2 minutes, your site will be at:

https://yourusername.github.io/home-grown

Updating Your Site

Whenever you want to update the site, just upload a new index.html to the same repo and it will automatically redeploy within minutes.

Custom Domain (Optional)

You can use a custom domain like homegrownsnacks.co.uk. In GitHub Pages settings, enter your domain name, then update your domain's DNS settings to point to GitHub. GitHub even provides a free SSL certificate.

Adding Stripe Payments

For real payments, you'll need a small backend server to create Stripe PaymentIntents. Good free options are:

  1. Netlify Functions — free serverless functions, works great with GitHub
  2. Vercel — free tier, easy Node.js backend
  3. Railway — simple Node.js/Python server hosting

Once your backend is running, paste the URL into Admin → Settings → Backend Payment URL.
